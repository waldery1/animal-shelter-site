## Тематика сайта, название
Сайт локального приюта для животных «Доброе сердце». Помощь в поиске новых хозяев для питомцев и сбор средств на их содержание.

## Цель и задачи сайта
Повышение уровня ответственности горожан по отношению к бездомным животным и упрощение процесса усыновления (адопции) питомцев.

## Целевая аудитория
Люди, желающие завести домашнее животное, волонтеры и жители города, готовые оказывать благотворительную помощь.

## Содержание сайта
* Главная страница с призывом к действию;
* Каталог животных с фотографиями и описанием;
* Страница «Как помочь» (реквизиты и список нужд);т 
* Контакты и форма обратной связи.

## Функциональные возможности
* Фильтрация животных по виду (кошки/собаки);
* Интерактивная форма для записи на посещение приюта;
* Адаптивная верстка для удобного просмотра с мобильных устройств.
