## Тематика сайта, название
Сайт локального приюта для животных «Доброе сердце». Помощь в поиске новых хозяев для питомцев и сбор средств на их содержание.

## Цель и задачи сайта
Повышение уровня ответственности горожан по отношению к бездомным животным и упрощение процесса усыновления (адопции) питомцев.

## Целевая аудитория
Люди, желающие завести домашнее животное, волонтеры и жители города, готовые оказывать благотворительную помощь.

## Содержание сайта
* Главная страница с призывом к действию;
* Каталог животных с фотографиями и описанием;
* Страница «Как помочь» (реквизиты и список нужд);
* Контакты и форма обратной связи.

## Функциональные возможности
* **Фильтрация каталога:** интерактивный поиск и сортировка животных по виду (кошки/собаки) без перезагрузки страницы.
* **Интерактивная форма записи:** форма бронирования времени для посещения приюта с валидацией введенных данных на стороне клиента.
* **Сохранение данных:** запись информации из формы в память браузера без отправки на внешний сервер.
* **Панель заявок:** динамический вывод списка всех созданных пользователем записей на страницу с возможностью их удаления.
* **Адаптивность:** корректное отображение и удобная навигация на смартфонах, планшетах и компьютерах.
